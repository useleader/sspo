# 02_预备知识

## 2.1 LLM 对齐问题

### 问题定义
LLM对齐是指使语言模型输出符合人类偏好的过程。给定一个prompt $x$，人类偏好表示为：
- $y_w$: 被偏好的响应（winner）
- $y_l$: 不被偏好的响应（loser）

### 数学形式化
对齐优化目标是最大化人类偏好的概率：
$$\max_\theta \mathbb{E}_{(x, y_w, y_l) \sim \mathcal{D}} \left[ \log P_\theta(y_w|x) - \log P_\theta(y_l|x) \right]$$

## 2.2 偏好学习损失函数

### DPO (Direct Preference Optimization)
$$\mathcal{L}_{DPO} = -\mathbb{E}_{(x, y_w, y_l) \sim \mathcal{D}} \left[ \log \sigma \left( \beta \log \frac{P_\theta(y_w|x)}{P_{ref}(y_w|x)} - \beta \log \frac{P_\theta(y_l|x)}{P_{ref}(y_l|x)} \right) \right]$$

其中：
- $\beta$: 温度参数，控制偏离参考模型的程度
- $\sigma$: sigmoid函数
- $P_{ref}$: 参考模型（通常为SFT后的模型）

### KTO (Kahneman-Tversky Optimization)
$$\mathcal{L}_{KTO} = -\mathbb{E}_{(x, y_w, y_l) \sim \mathcal{D}} \left[ \log \sigma \left( \beta \log \frac{P_\theta(y_w|x)}{P_{ref}(y_w|x)} - \log \frac{P_\theta(y_l|x)}{P_{ref}(y_l|x)} \right) \right]$$

### ORPO (Odds Ratio Preference Optimization)
$$\mathcal{L}_{ORPO} = \mathcal{L}_{SFT} + \alpha \cdot \mathbb{E}_{(x, y_w, y_l)} \left[ \log \sigma \left( \frac{P_\theta(y_w|x)}{P_\theta(y_l|x)} \right) \right]$$

## 2.3 半监督学习基础

### 核心思想
利用少量标注数据和大量无标注数据共同训练模型。

### 关键技术
1. **伪标签(Pseudo-labeling)**: 用当前模型对无标注数据生成标签
2. **一致性正则化(Consistency Regularization)**: 鼓励模型对相似样本输出一致结果
3. **熵最小化(Entropy Minimization)**: 鼓励模型对无标注数据给出高置信度预测

### 传统SSL方法
| 方法 | 原理 | 适用场景 |
|------|------|----------|
| Self-Training | 用模型预测作为伪标签重新训练 | 分类任务 |
| Co-Training | 多视图互补学习 | 多特征场景 |
| MixUp | 插值增强 | 图像/文本 |

## 2.4 论文相关工作对比

### 端到端对齐方法
| 方法 | 核心思想 | 优势 | 局限 |
|------|----------|------|------|
| RLHF | 使用reward模型+PPO | 理论完善 | 训练不稳定 |
| DPO | 直接优化偏好差异 | 无需reward模型 | 需要大量标注 |
| KTO | 基于前景理论 | 减少幻觉 | 需要大量标注 |
| ORPO | odds ratio优化 | 简洁高效 | 需要大量标注 |
| **SSPO** | **半监督+伪标签** | **数据高效** | **依赖伪标签质量** |

## 2.6 论文核心假设

1. **特征一致性假设**: 相似输入应产生相似偏好标签
2. **伪标签可靠性假设**: 高置信度伪标签与真实标签大概率一致
3. **分布平滑假设**: 偏好数据的决策边界应较为平滑
