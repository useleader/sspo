# 06_实验设置

## 6.1 数据集

### 6.1.1 Paired Data (偏好标注数据)

| 数据集 | 大小 | 领域 | 用途 |
|--------|------|------|------|
| UltraFeedback | 61K | 通用 | 主实验 |
| UltraMedical-Preference | ~11K | 医疗 | 领域泛化 |
| DSP Business | ~1.2K | 商业 | 领域泛化 |

### 6.1.2 Unpaired Data (无标注SFT数据)

| 数据集 | 大小 | 领域 | 用途 |
|--------|------|------|------|
| UltraChat-200k | 20K | 通用 | 主实验unpaired |
| UltraMedical | 20K | 医疗 | 医疗领域 |
| 17k Business Book | 17K | 商业 | 商业领域 |

### 6.1.3 数据划分策略
- **1% setting**: $n_L \approx 611$ (UltraFeedback)
- **10% setting**: $n_L \approx 6,113$ (UltraFeedback)

## 6.2 基线模型

| 方法 | 描述 | 数据需求 |
|------|------|----------|
| DPO | Direct Preference Optimization | 100% |
| ORPO | Odds Ratio Preference Optimization | 100% |
| SimPO | Simplified Preference Optimization | 100% |
| KTO | Kahneman-Tversky Optimization | 100% |
| SSRM | Semi-Supervised Reward Modeling | 半监督 |
| SPA | Spread Preference Annotation | 迭代自标注 |
| **SSPO** | **本文方法** | **半监督** |

## 6.5 超参数

### 训练超参数
| 参数 | 值 |
|------|-----|
| learning_rate | 5e-7 |
| batch_size | per device 8 |
| num_train_epochs | 3 |
| max_seq_length | 2048 |

### SSPO特定参数
| 参数 | 值 | 说明 |
|------|-----|------|
| Prior $\mathbb{P}_{D_U}(s=1)$ | 0.5 | balanced prior |
| $\lambda$ (decay rate) | - | 自适应调度 |
| $\gamma_0$ | 1 | 初始系数 |
| $\gamma_{min}$ | $n_L/(n_L+n_U)$ | 最小系数 |

## 6.6 实验设计

### Toy Experiment
- 使用NLTK词长作为偏好信号（最短词=winning，最长词=losing）
- 测试不同$n_L$ (10, 50, 100)和噪声级别
- 使用GPT-2-small作为reward model

### Real Data Experiment
- 在3个领域（通用、医疗、商业）测试
- 对比1%和10%标注比例下的性能
- 消融分析prior sensitivity和adaptive scheduling
