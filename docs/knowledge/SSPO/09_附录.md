## A.1 Theorem 1详细证明

### 证明
设$r_l^{(i)} := r_\theta(x^{(i)}, y_l^{(i)})$，$r_w^{(j)} := r_\theta(x^{(j)}, y_w^{(j)})$。

由sub-Gaussian性质，对任意$t_1 > 0$：
$$\mathbb{P}(r_l^{(i)} > \mu_l + t_1) \leq \exp(-t_1^2/2\sigma_l^2)$$

由union bound：
$$\mathbb{P}(\max_i r_l^{(i)} > \mu_l + t_1) \leq n_L \cdot \exp(-t_1^2/2\sigma_l^2)$$

类似地：
$$\mathbb{P}(\min_j r_w^{(j)} < \mu_w - t_2) \leq n_L \cdot \exp(-t_2^2/2\sigma_w^2)$$

设$\delta^* = \mu_l + t_1 = \mu_w - t_2$，由假设$\mu_w > \mu_l$知$\delta^*$存在。

令$\alpha \geq n_L \cdot (e^{-t_1^2/2\sigma_l^2} + e^{-t_2^2/2\sigma_w^2})$，则：
$$\mathbb{P}(\max_i r_l^{(i)} \leq \delta^* \leq \min_j r_w^{(j)}) \geq 1 - \alpha$$

QED.

## A.2 伪标签风险的实现细节

对于unpaired data $D_U$，pseudo-labeled risk为：
$$R_{D_U}(f_\theta) = \frac{1}{n_U}\sum_{k=1}^{n_U}\ell(f_\theta, \tilde{s}_k) \cdot \mathbb{P}_{D_U}(s=\tilde{s}_k)$$

其中$\tilde{s}_k = \mathbb{I}\{r_\theta(x_u^{(k)}, y_u^{(k)}) > \hat{\delta}\}$。

## A.3 梯度分析

$$\nabla_\theta R_{D_L} = \mathbb{E}_{D_L}[\nabla_\theta (-\log\sigma(\Delta r - \Delta))]$$

$$\nabla_\theta R_{D_U} = \mathbb{E}_{D_U}[\nabla_\theta \ell(f_\theta, \tilde{s})]$$

总梯度：
$$\nabla_\theta \mathcal{L} = \gamma' \cdot \nabla_\theta R_{D_L} + (1-\gamma') \cdot \nabla_\theta R_{D_U}$$

## A.4 算法伪代码

```python
def SSPO(D_L, D_U, config):
    # D_L: [(x, y_w, y_l), ...]
    # D_U: [(x, y), ...]

    # 初始化
    θ = initialize_from_sft()
    γ_min = len(D_L) / (len(D_L) + len(D_U))
    γ' = 1.0

    for epoch in range(num_epochs):
        # 1. 从paired data学习reward function
        r_θ = reward_model(D_L, θ)

        # 2. 用KDE估计threshold
        δ̂ = estimate_threshold_kde(r_θ, D_L)

        # 3. 为unpaired data生成伪标签
        pseudo_data = []
        for (x_u, y_u) in D_U:
            if r_θ(x_u, y_u) > δ̂:
                pseudo_data.append((x_u, y_u, 'pseudo-winning'))
            else:
                pseudo_data.append((x_u, y_u, 'pseudo-losing'))

        # 4. 自适应调度
        γ' = max(γ_min, exp(-λ * epoch))

        # 5. 联合优化
        loss = γ' * R_D_L(θ) + (1-γ') * R_D_U(θ)
        θ = θ - lr * gradient(loss, θ)

    return θ
```

## A.7 局限性

1. **Reward分布假设**: sub-Gaussian假设在实践中可能不成立
2. **分布偏移**: paired/unpaired数据分布可能不同
3. **计算开销**: KDE估计增加了计算成本
4. **Prior敏感性**: 虽然0.5最优，但需要调优

## A.8 未来方向

1. **自适应Prior**: 根据伪标签质量动态调整
2. **多任务SSL**: 同时利用多种无标注信号
3. **在线学习**: 边生成伪标签边训练
4. **理论改进**: 放松sub-Gaussian假设
