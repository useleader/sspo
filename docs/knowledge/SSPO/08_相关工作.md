# 08_相关工作

## 8.1 Preference Optimization方法分类

```
┌─────────────────────────────────────────────────────────────────┐
│                    Preference Optimization                        │
├─────────────────────────────────────────────────────────────────┤
│                                                                  │
│  RL-based                                                         │
│  └── RLHF (PPO) - 需要reward model，训练复杂                     │
│                                                                  │
│  Direct Optimization                                              │
│  ├── DPO - 将偏好学习重构为二元分类                              │
│  ├── ORPO - 使用odds ratio                                       │
│  ├── SimPO - 简化版，移除reference model                         │
│  └── KTO - 基于前景理论                                          │
│                                                                  │
│  Semi-supervised (本文)                                           │
│  ├── SSRM - 自训练的reward modeling                              │
│  ├── SPA - 迭代自标注                                             │
│  └── SSPO - 基于Bayes最优threshold的伪标签                        │
│                                                                  │
└─────────────────────────────────────────────────────────────────┘
```

## 8.2 各方法对比

| 方法 | 需要Reference | 需要Reward | 标注需求 | 半监督 |
|------|--------------|------------|----------|--------|
| RLHF | ❌ | ✅ | 100% | ❌ |
| DPO | ✅ | ❌ | 100% | ❌ |
| ORPO | ❌ | ❌ | 100% | ❌ |
| SimPO | ❌ | ❌ | 100% | ❌ |
| KTO | ✅ | ❌ | 100% | ❌ |
| SSRM | ✅ | ✅ | 半监督 | ✅ |
| SPA | - | - | 迭代自标注 | ✅ |
| **SSPO** | ✅ | ❌ | **半监督** | ✅ |

## 8.3 SSPO vs SSRM

| 方面 | SSRM | SSPO |
|------|------|------|
| 伪标签对象 | 用于训练reward model | 直接用于policy training |
| 理论基础 | self-training | Bayes最优分类 |
| 阈值 | 固定或启发式 | KDE估计+Bayes risk最小化 |
| 调度 | 固定权重 | 自适应curriculum |

## 8.4 SSPO vs SPA

| 方面 | SPA | SSPO |
|------|-----|------|
| 策略 | 迭代自标注 | reward threshold伪标签 |
| 计算成本 | 高（多次生成） | 低（一次性KDE估计） |
| 理论基础 | 无明确理论 | Theorem 1证明存在性 |

## 8.5 SSPO的独特贡献

1. **理论框架**: 首次将偏好学习重构为Bayes最优分类问题
2. **最优阈值存在性**: 证明存在$\delta^*$高概率分离winning/losing
3. **实用阈值估计**: 使用KDE从paired data学习threshold
4. **自适应调度**: curriculum learning平衡paired/unpaired数据
