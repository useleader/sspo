# 03_问题定义

## 3.1 符号系统

### 基础符号

| 符号 | 定义 |
|------|------|
| $x$ | 输入prompt |
| $y_w$ | 被偏好的响应 (winner) |
| $y_l$ | 不被偏好的响应 (loser) |
| $\theta$ | 待优化模型参数 |
| $\phi$ | 参考模型参数 |
| $\beta$ | 温度/kl散度权重参数 |

### 数据集符号

| 符号 | 定义 | 数量级 |
|------|------|--------|
| $\mathcal{D}_l$ | 标注偏好数据集 | ~10K |
| $\mathcal{D}_u$ | 无标注偏好数据集 | ~100K |
| $n_l = \|\mathcal{D}_l\|$ | 标注样本数 | 2K-10K |
| $n_u = \|\mathcal{D}_u\|$ | 无标注样本数 | 20K-100K |

### 概率符号

| 符号 | 定义 |
|------|------|
| $P_\theta(y\|x)$ | 模型对响应y的条件概率 |
| $P_\phi(y\|x)$ | 参考模型对响应y的条件概率 |
| $\hat{y}$ | 伪标签（模型预测的偏好响应） |

## 3.2 问题形式化

### 传统偏好优化（监督学习设定）

给定标注数据集 $\mathcal{D}_l = \{(x_i, y_w^i, y_l^i)\}_{i=1}^{n_l}$，优化目标：

$$\min_\theta \mathcal{L}_{sup}(\theta) = \sum_{i=1}^{n_l} \ell_{pref}(x_i, y_w^i, y_l^i; \theta)$$

其中 $\ell_{pref}$ 可以是 DPO、KTO、ORPO 等损失函数。

### 半监督偏好优化（新设定）

给定：
- 少量标注数据集 $\mathcal{D}_l$（$n_l$ 样本）
- 大量无标注数据集 $\mathcal{D}_u$（$n_u$ 样本，$n_u >> n_l$）

目标：利用 $\mathcal{D}_l \cup \mathcal{D}_u$ 优化 $\theta$，性能接近使用全量标注数据。

## 3.3 无标注数据的结构

### 偏好对的表示
无标注数据 $\mathcal{D}_u$ 中每条记录包含：
$$\{x_j, (y_j^1, y_j^2)\}$$

即一个prompt和两个候选响应，但不知道哪个更好。

### 关键挑战
1. **谁更优未知**: 不确定 $y^1$ 还是 $y^2$ 更符合人类偏好
2. **候选顺序未知**: 不知道哪个响应是第一个位置
3. **需要伪标签**: 必须先推断哪个响应是preferred

## 3.4 优化目标

### 最终目标

$$\min_\theta \mathcal{L}_{total}(\theta) = \mathcal{L}_{sup}(\theta) + \lambda \cdot \mathcal{L}_{ss}(\theta)$$

其中：
- $\mathcal{L}_{sup}$: 监督损失（来自 $\mathcal{D}_l$）
- $\mathcal{L}_{ss}$: 半监督损失（来自伪标签数据）
- $\lambda$: 平衡权重

### 伪标签生成目标

对每个无标注样本 $(x, y^1, y^2) \in \mathcal{D}_u$：

$$\hat{y} = \arg\max_{y \in \{y^1, y^2\}} P_\theta(y|x)$$

即选择模型认为更好的响应作为伪标签。

## 3.5 关键约束

### 伪标签质量约束

$$\mathbb{E}[\mathbf{1}(\hat{y} \neq y^*)] \leq \epsilon$$

其中 $y^*$ 是真实偏好，$\epsilon$ 是可容忍的伪标签错误率上限。

### 分布匹配约束

$$d(\mathcal{D}_l, \hat{\mathcal{D}}_u) \leq \delta$$

确保伪标签数据的特征分布与真实标注数据接近。

## 3.7 与其他问题的区别

| 问题 | 特点 | SSPO适用性 |
|------|------|-----------|
| 文本分类 | 类别固定 | ✅ 可迁移 |
| 情感分析 | 二分类 | ✅ 可迁移 |
| 偏好预测 | 成对排序 | ✅ 直接适用 |
| 强化学习 | 序列决策 | ⚠️ 需调整 |
| 排序学习 | 查询相关 | ✅ 可迁移 |

## 3.8 评估指标

### 偏好准确率
$$Acc = \frac{1}{|\mathcal{D}_{test}|} \sum_{(x, y_w, y_l) \in \mathcal{D}_{test}} \mathbf{1}\left( \arg\max_{y \in \{y_w, y_l\}} P_\theta(y|x) = y_w \right)$$

### KL散度约束
$$\mathbb{E}_x [D_{KL}(P_\theta(\cdot|x) \| P_\phi(\cdot|x))] \leq \beta$$

### 伪标签质量
$$Precision_{pl} = \frac{| \{ (x, \hat{y}) : \hat{y} = y^* \} |}{|\hat{\mathcal{D}}_u|}$$
